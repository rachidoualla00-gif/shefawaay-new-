# Shefaaway Islamic App

A premium, modern, and minimalist Islamic mobile application tailored for Arabic-first users. Built with React Native (Expo).

## 🌟 Overview
Shefaaway provides an elegant and spiritual user experience featuring:
- **الرئيسية (Home)**: Islamic dashboard with next prayer countdown, daily Hadith, and Duas.
- **القرآن (Quran)**: Built-in PDF reader with page tracking capabilities.
- **الدروس (Doros)**: Video lessons and Islamic lectures.
- **الصلاة (Prayer)**: Location-based accurate prayer times and Qibla compass.
- **المتجر (Store)**: Premium wellness and lifestyle product showcase.

## 🛠️ How to View Locally (No Code Required)
If you want to view the app on your own mobile device without installing any developer tools:

1. Download the **Expo Go** app on your iOS or Android device.
2. Open your device's camera and scan the QR code generated when you run the app (details below), or click the published Expo Link if provided.

## 💻 Developer Setup Instructions
If you wish to run the app in a local development environment:

1. **Install Node.js** (v18 or newer recommended).
2. **Clone this repository** to your machine.
3. Open a terminal inside the project folder and install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npx expo start
   ```
5. Press `w` in the terminal to open the web version, or scan the QR code with your phone using the Expo Go app.
